# Desafio your/dev

 Desafio Soptify desenvolvido para a vaga de Dev Web na your/dev.

## Instalação

### 1. Clone o Repositório

```sh
git clone https://github.com/HerbertVeloso/desafio-yourdev.git
cd desafio-yourdev
```

### 2. Instale as Dependências & Rode Localmente

```sh
yarn install
yarn start
```
