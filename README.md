# Desafio your/dev

 Desafio Soptify desenvolvido para a vaga de Dev Web na your/dev.

## Instalação

### 1. Clone o repositório

```sh
git clone https://github.com/HerbertVeloso/desafio-yourdev
cd desafio-yourdev
```

### 4. Instale as dependências & rode localmente

```sh
yarn install
yarn start
```
